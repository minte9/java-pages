## Password Strength Checker (Security-Oriented)

### What it does

- Reads passwords from input
- Validates:
   - Length
   - Upper/lowercase
   - Digits
   - Special characters
   - Scores password strength

### Concepts used

      Regexp (core focus)
      Strings
      Static utility classes
      Interfaces (rules engine)
      Polymorphism (different validators)
      Exceptions
      JUnit parameterized tests
