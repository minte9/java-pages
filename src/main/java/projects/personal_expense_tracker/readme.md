## Personal Expense Tracker (CLI)

### What it does

- Add expenses
- List expenses
- Filter by category / date
- Show totals

### Concepts used

    Classes, Constructors
    Lists, Maps
    Streams (filter, reduce)
    Enums (categories)
    Properties file (storage path, currency)
    JUnit tests

### Sample commands

    add 2025-11-20 FOOD 12.50
    list FOOD
    total MONTH
